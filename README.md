This is a simple *Online Auction System* developed in Java using core *Data Structures and Algorithms (DSA)* concepts.  
The system allows users to register bidders, add auction items, place bids, and track the highest bid using a *Priority Queue (Max Heap)*.

---

## Features
- Add auction items
- Register bidders
- Place bids on items
- Display current highest bid
- End auction and declare winner
- View bidding history
- Search auction items by ID or name

---

## DSA Concepts Used
- Priority Queue (Max Heap)
- ArrayList
- HashMap
- Searching
- Sorting

---

## Classes Used
- AuctionItem
- Bidder
- Bid
- AuctionSystem (Main Class)

---

## How It Works
1. Admin adds auction items.
2. Users (bidders) are registered in the system.
3. Bidders place bids on items.
4. System stores bids using a Max Heap (highest bid gets priority).
5. User can view:
   - Current highest bid
   - Full bidding history
   - Winner after auction ends
